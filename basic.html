All BASIC IN HTML

->html: This is the root element of an HTML document.
->head: Contains meta-information about the HTML document (e.g., title, charset).
->title: Sets the title of the HTML document, which appears in the browser's title bar or tab.
->body: Contains the content of the HTML document.
->h1 to h6: Heading tags, with h1 being the highest level and h6 the lowest.
->p: Paragraph tag.
->a: Anchor tag for creating hyperlinks.
->img: Embeds an image.
->div: Defines a division or section in the document.
->span: Inline container for text and other inline elements.
->ul and li: Unordered list and list item.
